Google Sheets Integration

📊 Sync Job Data to Google Sheets
When configured, completed Vactor jobs will automatically sync to a Google Sheet where supervisors and engineers can view all job data in real-time.
See setup instructions below. Leave blank to save data locally only.

Status

No webhook configured - data saves to device only

Setup Instructions

Google Sheets Setup Guide

Step 1: Create Google Sheet

  1. Go to sheets.google.com
  2. Create a new blank spreadsheet
  3. Name it "WWTP Vactor Jobs" or similar

Step 2: Set Up Apps Script

  1. In your Google Sheet, click Extensions → Apps Script
  2. Delete any existing code
  3. Copy and paste the script code (see below)
  4. Click Save (disk icon)
  5. Click Deploy → New deployment
  6. Click gear icon ⚙️, select Web app
  7. Set "Execute as" to Me
  8. Set "Who has access" to Anyone
  9. Click Deploy
  10. Copy the Web app URL (this is your webhook URL)

Step 3: Configure App

  1. Go back to Settings in this app
  2. Paste the webhook URL into the field
  3. Click Test Connection to verify
  4. Click Save Settings
⚠️ Important Notes
  • Anyone with the webhook URL can send data to your sheet
  • Don't share the URL publicly
  • You can regenerate a new deployment if needed

Apps Script Code

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // Create headers if this is the first row
  if (sheet.getLastRow() === 0) {
    sheet.appendRow([
      'Timestamp',
      'Date', 
      'Time',
      'Operator',
      'Team',
      'Location',
      'Duration (min)',
      'Decants',
      'Dumps',
      'Fills',
      'Total Operations',
      'Job ID'
    ]);
  }
  
  // Parse the incoming data
  var data = JSON.parse(e.postData.contents);
  
  // Append new row
  sheet.appendRow([
    data.timestamp,
    data.date,
    data.time,
    data.operator,
    data.team,
    data.location,
    data.durationMinutes,
    data.decantCount,
    data.dumpCount,
    data.fillCount,
    data.totalOperations,
    data.jobId
  ]);
  
  return ContentService.createTextOutput('Success');
}
✓ After Setup
Every time an operator completes a job, the data will automatically appear as a new row in your Google Sheet. Supervisors and engineers can access the sheet from any device with their Google account.
